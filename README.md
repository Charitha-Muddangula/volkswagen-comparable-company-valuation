# Comparable Company Valuation – Volkswagen AG

## 📌 Overview

This project performs a relative valuation of Volkswagen AG using Comparable Company Analysis (Comps).

The model benchmarks Volkswagen’s valuation against global automotive peers using EV/EBITDA multiples.

---

## 🎯 Objective

To determine whether Volkswagen AG is relatively overvalued or undervalued compared to its peer group.

---

## 🧩 Key Features

* Peer group selection (BMW, Ford, Mercedes)
* EV/EBITDA multiple comparison
* Relative valuation benchmarking
* Scenario-based valuation range analysis

---

## 📊 Key Insight

The analysis highlights Volkswagen’s valuation positioning relative to peers and identifies potential undervaluation based on multiple comparison.

---

## 🛠 Tools Used

* Microsoft Excel
* Comparable Company Analysis (Comps)
* Financial data from Yahoo Finance and Corporate Annual Reports

---

## 📂 File Included

* `Volkswagen Project_1.xlsx` – Complete valuation model

---

## 🚀 How to Use

1. Open the Excel file
2. Review peer multiples and valuation inputs
3. Adjust assumptions if required
4. Analyze valuation range and relative positioning
